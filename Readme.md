# NFT Fractionalization

Create a pool from the NFT you have received. 
Fractionalize your NFT by creating a pool.
Distribute the pool tokens to your audience to let them join your room.

### How to set up?

Client setup:

```bash
cd client
# create and addd env variables
vim .env.local
# add variables
MONGO_URI=
MONGO_DB=
# start server
yarn
yarn run dev
```

Backend setup:
```bash
cd backend
# create and addd env variables
vim .env
# add variables
DB_USER=
DB_PASSWORD=
DB_URL=
DB_NAME=
# start server
docker build -t back .
docker run -p 80:80
```
